# EC-Clustering
Code and datasets for article ''EC-Clustering: A Federated Clustering Algorithm For Edge-Cloud Collaborative Intelligence''
## Dependencies
- Python (>= 3.6)
- NumPy (>= 1.13.3)
- SciPy (>= 0.19.1)
- Sklearn (>= 0.11)

 |non-IID partitioned ImagenetS dataset   |ImagenetS_noniid_edge_number.csv   |
